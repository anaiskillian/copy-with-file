# Copy With File

Copy the current selection (or line) along with the full file path and line number.

## Features

- `cmd + shift + P`: `Copy With File (code, file, line)`
- Output format:
  ```bash
  (
  <code>,
  "/full/path/to/file",
  line <n>
  )
  ```

## Default Keybinding

- macOS: `cmd + option + c`
- You can change this in your own keybindings.
