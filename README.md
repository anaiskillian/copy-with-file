# Copy With File

Copy the current selection (or line) along with the full file path and line number.

## Features

- Command: `Copy With Context (code, file, line)`
- Output format:
  ```
  (
  <code>,
  "/full/path/to/file",
  line <n>
  )
  ```

## Default Keybinding

- macOS: `cmd+alt+c`
- You can change this in your own keybindings.
