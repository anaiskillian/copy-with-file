# Copy With File

Copy the current selection (or line) along with the full file path and line number.

## Features

- `cmd + shift + P`: `Copy With File (code, file, line)`

Output format:
```bash
<code>,
"/full/path/to/file",
line <n>
```

Example:
```bash
data = [5, 1, 4, 2, 8],
"/Users/dir/file.py",
line 26
```

## Default Keybinding

- macOS/ windows/ linux: : `cmd + option + c`
- You can change this in your own keybindings.
